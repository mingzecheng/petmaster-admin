# 宠物管理系统 - 启动指南

## 🚀 快速启动

### 1. 启动后端服务

```bash
cd /Users/mingzechen/PycharmProjects/PetMaster/PetMaster-fastapi
python main.py
```

后端服务将运行在 http://localhost:8000

### 2. 启动前端服务

```bash
cd /Users/mingzechen/PycharmProjects/PetMaster/petmaster-admin
npm install  # 首次运行需要安装依赖
```

前端服务将运行在 http://localhost:3000

打开浏览器访问: http://localhost:3000

## ⚙️ 配置说明

### 后端API地址配置

前端通过 Vite 代理配置连接后端:

```typescript
// vite.config.ts
server: {
  port: 3000,
  proxy: {
    '/api': {
      target: 'http://localhost:8000',
      changeOrigin: true,
    },
  },
}
```

### 后端CORS配置

确保后端 `main.py` 中的CORS配置包含前端地址:

```python
app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:3000"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```

## 📝 开发说明

### 添加新页面

1. 在 `src/views/` 创建页面组件
2. 在 `src/router/index.ts` 添加路由
3. 在 `src/api/` 添加对应的API接口
4. 在 `src/types/` 添加类型定义

### 权限控制

在路由meta中配置角色:

```typescript
{
  path: 'users',
  meta: { title: '用户管理', roles: ['admin'] }
}
```

## 🐛 常见问题

### 1. 无法连接后端

- 检查后端服务是否启动
- 检查端口是否被占用
- 查看浏览器控制台错误信息

### 2. 登录失败

- 确认数据库中有对应用户
- 检查用户名密码是否正确
- 查看后端日志

### 3. 安装依赖失败

- 尝试使用 `npm install --legacy-peer-deps`
- 或使用 `yarn` 或 `pnpm` 安装
